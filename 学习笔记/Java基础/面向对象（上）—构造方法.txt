               构造方法
类的特殊成员，在实例化对象时被自动调用。条件：
①方法名与类名相同
②方法名的前面没有返回值类型的声明
③在方法中不使用return语句返回值	
如：class Person{
     //下面是类的构造方法
        public Person(){
		     System.out.println(" ");
			 }
    }		 
无参构造方法、有参构造方法	
构造方法重载：一个类中可以定义多个构造方法，多个方法的参数类型或参数个数不同即可
java的每个类中至少有一个构造方法，如果没有定义构造方法，系统会默认创建一个空的构造方法
其方法体为空，如：class Person{};或class Person{public Person(){}}

构造方法一般使用public
