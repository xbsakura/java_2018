     面向对象（上）
面向对象的三个特点：封装性（核心）、继承性、多态性
简单来说，类是对象的模板（抽象），对象是类的实例
在类中可以定义1、成员变量（属性），2、成员方法

成员变量：在类中定义的变量；
局部变量：在方法中定义的变量；
java中允许在某个类的一个方法中定义的局部变量和成员变量同名，此时在方法中通过变量名访问到的是局部变量而非成员变量

使用new关键字来创建一个对象:类名 对象名称=new 类名();
对象的引用来访问对象成员：对象引用.对象成员

实例化对象时，java虚拟机自动为成员变量初始化
类型    初始值
byte     0
short    0
int      0
long     0L
float    0.0F
double   0.0D
char     空字符，'\u0000'
boolean  false
引用类型 null

类的封装：定义类时，将类中的属性私有化。
