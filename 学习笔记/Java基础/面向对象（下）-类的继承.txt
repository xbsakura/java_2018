                       类的继承
继承:在一个现有类（父类）的基础上构建一个新的类（称为子类），子类自动拥有父类所有可继承的属性和方法。
声明：extends关键字
注意：1、在Java中，类只支持单继承，不允许多重继承（一个类只能有一个直接父类），如下面
class A{}
class B{}
class C extends A,B{}（C不能同时继承A和B）
2、多个类可以继承一个父类，如
class A{}
class B extends A{}
class C extends A{}
3、Java允许多层继承，一个类的父类可以再去继承另外的父类，如
class A{}
class B extends A{}
class C extends B{}
4、子类和父类是一对相对概念（一个类是某个类父类的同时，也可以是另一个类的子类）

重写父类方法：子类中重写的方法必须和父类中被重写的方法具有相同的方法名、参数列表、返回值类型
注意：重写父类方法时，不能使用比父类更严格的访问权限（父类是public，子类就不能是private）。

super关键字：为了解决重写父类方法后，子类对象无法访问父类被重写的方法。
①调用父类成员变量和成员方法
super.成员变量
super.成员方法([参数1.参数2...])
②调用父类的构造方法
super([参数1,参数2...])
注意：super调用父类构造方法的代码必须位于子类构造方法的第一行，且只能出现一次！
